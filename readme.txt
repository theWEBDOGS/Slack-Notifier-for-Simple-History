Slack notifications for SimpleHistory helps developers maintain continuity with their sites.


== Description ==

Selected events are posted to the [Slack](https://slack.com/) channels of your choice, using [incoming webhook](https://api.slack.com/incoming-webhooks). With this plugin enabled,
you can leverage Slack to see what is happening on any sites you are using Simple History.


== Screenshots ==

1. Enable the loggers you want in Settings » Simple History » Notifier.
